
# Aaron E. Reinick — Cost Optimization Strategist Landing Page

A polished, single-page site in a clean white & deep blue executive style, optimized for trust and conversion.

## Visual Direction
- **Palette:** crisp white background, deep navy (#0B2545-ish) primary, steel-blue accents, subtle gray section dividers, gold/amber highlight reserved for key stats.
- **Typography:** modern serif for headlines (authority), clean sans-serif for body (clarity).
- **Feel:** McKinsey/Bain-inspired — generous whitespace, confident typography, restrained motion, sharp data callouts.

## Functional / Backend
- **Lovable Cloud** enabled to host the email-sending edge function.
- **Lovable Email** infrastructure used to send each contact form submission to `areinick@eragroup.com` (one email per submission, includes name/email/company/message). User sees a success toast on submit.
- All other content is static, fully responsive (mobile, tablet, desktop), with semantic HTML and accessible contrast.
